## vue3移动端组件库，整体风格类似vant组件库

### 安装

```
npm install lz-ui
```

### 使用

#### 1.全量使用 main.js中导入

```
import LZUI from 'lz-ui'
import 'lz-ui/es/style.css'

app.use(LZUI)
```

#### 2.按需加载 用什么导入什么，样式需要先引入到main.js

```
import 'lz-ui/es/style.css'
```
